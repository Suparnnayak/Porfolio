# Suparn Nayak Portfolio

Personal portfolio site built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Highlights

- Neo-brutalist portfolio UI with animated sections and custom components
- Personalized project case studies for `AidLink`, `HealthFlow AI`, `IdeaLens AI`, and `SkyRescue Vision`
- Stack, focus areas, and contact pages aligned with Suparn Nayak's resume
- Multi-step contact form wired through the API route in `app/api/contact/route.ts`

## Tech

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
